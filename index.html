This project can be used to configure your own NR/LTE network. If gNB/eNB and USIM are available,
you can build a private network using Open5GS.

Open5GS implemented 5GC and EPC using C-language. And WebUI is provided for testing purposes
and is implemented in Node.JS and React.

For more information, visit to our blog :

  • https://open5gs.org/open5gs
